# Intended user(s) of the code for this repo
The user(s) who are provided access to this repo are those trusted by either the owner or a trusted individual with access.

# Assessment of the risk of security threats
If code from this repo is accessed by a malicious user, then it should be determined what sensitive information is leaked and how the code in the repo can be used in an unintended malicious way.
There is the posibility that sensitive information, like an individual's name, is given to a malicious user. Also, exisitng code or new code can be introduced into the repo as malware by a malicious 
user, which could harm trusted users in the repo who assume that all code in the repo is safe.

# Steps taken to secure this repo
To secure this repo, its privacy settings have been modified to become a private repo instead of a public repo. This makes the repo only viewable by individuals that are given permission. This also
prevents any user outiside the trusted group from making changes to the repo or forking the code from the repo.
